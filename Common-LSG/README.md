# Common-LSG 层级截面几何通用底层核心库

## 理论归属
本模块为 LSG (Layered Section Geometry) 层级截面几何的理论本体，
是 GYYG-OES 源演体系在几何工程领域的核心落地内核。

## 公有领域声明
本目录下全部基础几何算法、截面矩阵、层级积分、维度互易变换、非正交切片重建逻辑，
永久归入人类公共知识领域，任何组织与个人不得基于本核心库申请独占性专利。

## 四大核心模块
### 1. axiom_calc 几何公理计算库
对应四条本源公理的数学实现：
- 空间承载：截面映射与三维空间基底定义
- 无限可分：层级梯度积分与多尺度拆分
- 维度互易：正负维度双向变换矩阵
- 对偶力驱动：梯度差与平衡约束计算

### 2. slice_rebuild 通用切片重建内核
全场景通用的非正交切片重建基础逻辑：
- 任意角度非正交切片空间对齐
- 多维度截面信息融合
- 通用层级积分三维重建

### 3. constraint_algo 基础约束算法
跨场景通用的几何校正与平滑能力：
- 通用噪声平滑
- 全局坐标校准
- 几何畸变基础修正

### 4. io_utils 通用IO工具
全行业兼容的数据输入输出接口：
- 通用传感器原始数据解析
- 点云模型导出
- STL 网格模型导出

### 核心求解器模块
- **core_gde_solver**：基于本源公理的全域对偶演化（GDE）LSG矩阵求解器，提供标准化三维微分算子、多层截面耦合、时域演化能力，是所有行业重建模块的统一数学底座。

## 调用规范
所有行业场景模块必须通过本底层库调用核心几何能力，
禁止重复实现基础截面、积分、变换逻辑，保证全体系理论同源。
